# 航空公司客户价值分析 (LRFMC 模型)

## ✈️ 项目简介
本项目通过对航空公司客户数据的探索性分析，利用 **LRFMC 模型** 进行聚类分群，从而识别出核心价值客户、潜在价值客户和易流失客户。

## 📊 技术栈
* **Python**: 数据处理核心
* **Pandas & NumPy**: 数据清洗与特征工程
* **Scikit-learn**: K-Means 聚类算法
* **Matplotlib/Seaborn**: 数据可视化（雷达图）

## 🔍 模型指标说明 (LRFMC)
* **L (Loyalty)**: 加入会员时长
* **R (Recency)**: 最近一次飞行距今时间
* **F (Frequency)**: 飞行次数
* **M (Mileage)**: 总飞行里程
* **C (Coefficient)**: 平均折扣率

## 📈 分析结论
通过聚类分析，我们将客户分为 5 类，并针对不同人群提出了针对性的营销建议（如对重要保持客户提供专属里程优惠等）。
